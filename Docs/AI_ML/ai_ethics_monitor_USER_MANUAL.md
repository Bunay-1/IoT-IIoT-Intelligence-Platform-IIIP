# Ръководство за потребителя: AI Ethics Monitor

## 1. Какво върши този модул?
Гарантира, че AI платформите се използват безопасно и честнo. Модулът филтрира опасни заявки към изкуствения интелект и проверява дали решенията на AI не дискриминират определени групи.

## 2. Начин на стартиране
```bash
python ai_ethics_monitor.py
```

## 3. Начин на работа
- **Safety Check**: Всяка заявка към LLM (езиков модел) преминава през филтър за токсичност.
- **Bias Analysis**: Модулът сравнява как AI третира различни групи (напр. 'Група А' срещу 'Група Б').

## 4. Интерпретиране на резултатите
- **Is Safe**: Ако е `False`, системата блокира заявката.
- **Max Disparity**: Разликата в отношението към различните групи. Стойности над 0.1 (10%) показват риск от пристрастие (bias).
- **Risk Level**: 'High' риск изисква човешка намеса и преглед на обучението на модела.
