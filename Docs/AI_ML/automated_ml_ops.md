# Документация: Automated MLOps

## Описание
Модулът `automated_ml_ops.py` реализира пълен MLOps жизнен цикъл, интегриран с MLflow. Той осигурява надеждност на моделите чрез проследяване на версиите и мониторинг на данните.

## Ключови функционалности
- **MLflow Tracking**: Автоматично логване на параметри, метрики и артефакти.
- **Детекция на дрейф на данни**: Използване на Колмогоров-Смирнов тест за откриване на промени в разпределението на входящите данни.
- **Автоматизирано преобучение**: Задействане на нов процес на обучение при откриване на значителен дрейф.
- **Управление на модели**: Преход на модели през различни етапи (Staging, Production, Archived).

## Употреба
```python
from automated_ml_ops import AutomatedMLOps

mlops = AutomatedMLOps(experiment_name="MyExperiment")
mlops.run_full_cycle(training_data, production_data)
```
