# Документация: AutoML Engine

## Описание
Модулът `automl_engine.py` предоставя усъвършенствани възможности за автоматизирано машинно обучение (AutoML). Той е проектиран да автоматизира жизнения цикъл на разработване на модели, включително инженерство на признаци, избор на модел и оптимизация на хиперпараметри.

## Ключови функционалности
- **Автоматично генериране на признаци**: Създаване на полиномиални признаци и взаимодействия (interactions) за подобряване на предсказателната сила.
- **Сравнение на множество модели**: Автоматично тестване на `LogisticRegression`, `RandomForest` и `GradientBoosting`.
- **Pipeline интеграция**: Използване на `scikit-learn Pipelines` за чиста и повторяема обработка на данни.
- **Leaderboard**: Генериране на класация на моделите въз основа на метрики за точност.

## Употреба
```python
from automl_engine import AdvancedAutoMLEngine

engine = AdvancedAutoMLEngine()
results = engine.run_automl(X, y)
print(results['leaderboard'])
```
