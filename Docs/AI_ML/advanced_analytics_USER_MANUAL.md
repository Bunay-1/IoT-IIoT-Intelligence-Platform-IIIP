# Ръководство за потребителя: Advanced Analytics

## 1. Какво върши този модул?
Модулът извършва комплексен статистически анализ на данни от сензори и машини. Той автоматично идентифицира скрити групи (клъстери), намалява сложността на данните (PCA) и открива аномални стойности, които могат да сигнализират за повреда.

## 2. Начин на стартиране
Модулът може да се стартира като самостоятелна демонстрация:
```bash
python advanced_analytics.py
```
Или да се интегрира в код:
```python
from advanced_analytics import AdvancedAnalytics
analytics = AdvancedAnalytics()
```

## 3. Начин на работа
1. **Зареждане на данни**: Подайте `pandas.DataFrame` с телеметрични данни.
2. **Анализ**: Извикайте `generate_analytics_report(df)`.
3. **Резултат**: Модулът ще върне речник с клъстери, аномалии и PCA вариация.

## 4. Интерпретиране на резултатите
- **Anomalies Found**: Брой записи, които се отклоняват значително от нормата. Ако броят е висок, проверете изправността на сензорите.
- **Cluster Distribution**: Показва как се групират машините. Различните клъстери могат да означават различни режими на работа.
- **PCA Variance**: Колкото по-близо е до 1.0 (100%), толкова по-добре PCA е запазил информацията при намаляване на размерността.
