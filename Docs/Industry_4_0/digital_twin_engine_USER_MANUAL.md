# Ръководство за потребителя: Digital Twin Engine

## 1. Какво върши този модул?
Създава "цифрово копие" на физически машини. Модулът следи здравето на машината в реално време и прогнозира **RUL (Remaining Useful Life)** – колко часа остават до следващата необходима поддръжка.

## 2. Начин на стартиране
```bash
python digital_twin_engine.py
```

## 3. Начин на работа
- **Създаване на двойник**: Дефинирайте актив (напр. "Robot-Arm-01").
- **Актуализация**: Подавайте данни за температура, вибрация и натоварване редовно.
- **Обучение**: След събиране на достатъчно данни (минимум 20 записа), модулът активира прогнозния си модел.

## 4. Интерпретиране на резултатите
- **Health Score (0-100)**: Всичко над 70 е "Healthy". Под 40 е "Critical" – опасност от внезапен отказ.
- **Predicted RUL**: Оставащи часове. Използвайте това число за планиране на техническо обслужване без спиране на производството.
