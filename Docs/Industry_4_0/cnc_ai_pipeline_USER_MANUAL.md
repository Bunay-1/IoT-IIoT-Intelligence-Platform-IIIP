# Ръководство за потребителя: CNC AI Pipeline

## 1. Какво върши този модул?
Специализирана система за управление на CNC машини. Тя автоматично оптимизира скоростта на рязане и консумацията на енергия, като същевременно следи износването на инструментите.

## 2. Начин на стартиране
```bash
python cnc_ai_pipeline.py
```

## 3. Начин на работа
Модулът работи като фонов процес (pipeline), който:
1. Събира телеметрия от шпиндела и осите.
2. Изчислява оптимални обороти.
3. Симулира износване и планира смяна на инструмент.

## 4. Интерпретиране на резултатите
- **Improvement Percentage**: Колко процента по-ефективна е станала машината след AI намесата.
- **Failure Probability**: Вероятност за счупване на инструмент в следващите часове.
- **Urgent Maintenance**: Ако е `True`, операторът трябва да спре машината веднага.
