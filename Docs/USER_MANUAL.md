# Глобално ръководство за потребителя (User Manual)
## IoT IIoT Intelligence Platform (IIIP) - v1.2.5

Добре дошли в официалното ръководство за потребителя на IIIP – интелигентна екосистема за индустриална автоматизация и изкуствен интелект.

---

## 1. Общ преглед на платформата
IIIP е проектирана да обслужва нуждите на съвременните заводи (Industry 4.0/5.0), предоставяйки инструменти за:
- Мониторинг в реално време на машини и активи.
- Прогностичен анализ и предсказваща поддръжка.
- Автоматизиран качествен контрол.
- Етичен надзор над AI алгоритмите.
- Проследяване на екологичното въздействие.

## 2. Инсталация и изисквания
### Системни изисквания
- **OS**: Linux / Windows (с Docker) / macOS
- **Python**: 3.11+
- **RAM**: Минимум 8GB (препоръчително 16GB за AI моделите)

### Бърза инсталация
```bash
git clone <repository-url>
cd mae-aiot-platform
pip install -r requirements.txt
```

## 3. Стартиране на платформата
### Вариант 1: Чрез Docker (Препоръчително)
```bash
docker-compose up --build
```
Това ще стартира ядрото на платформата и MLflow сървъра за проследяване.

### Вариант 2: Локално стартиране
```bash
python main.py
```

## 4. Структура на документацията
В директория `Docs/` ще намерите специфични ръководства за всеки ключов модул:
- `AI_ML/`: Аналитични инструменти и MLOps.
- `Industry_4_0/`: Качествен контрол и дигитални двойници.
- `Infrastructure/`: API Gateway и системни настройки.

## 5. Поддръжка и контакти
При възникване на проблеми, моля консултирайте се със съответното ръководство за модула или се свържете с администратора на системата.
