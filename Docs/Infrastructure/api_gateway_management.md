# Документация: API Gateway Management

## Описание
`api_gateway_management.py` действа като входна точка за всички външни заявки към микросервизите на платформата, осигурявайки сигурност и производителност.

## Ключови функционалности
- **Асинхронно проксиране**: Използване на `aiohttp` за ефективно препращане на заявки.
- **Интелигентно кеширане**: Вградена система за кеширане в паметта за намаляване на латентността.
- **Здравни проверки (Health Checks)**: Автоматичен мониторинг на състоянието на бекенд услугите.
- **Ограничаване на честотата (Rate Limiting)**: Защита от претоварване чрез интегриран Rate Limiter.

## Употреба
Стартиране през CLI:
```bash
python api_gateway_management.py --port 8080
```
