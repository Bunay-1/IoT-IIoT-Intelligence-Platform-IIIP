# Пътна карта (Roadmap) на IoT IIoT Intelligence Platform

Този документ очертава етапите на развитие на платформата, текущото състояние и бъдещите планове.

## 🟢 Фаза 1: Основи и Интеграция (Завършена)
- [x] Изграждане на скелет на платформата и главен оркестратор (`main.py`).
- [x] Реализиране на базови IoT и IIoT функционалности.
- [x] Интеграция с Kafka за стрийминг на данни.
- [x] Създаване на базов API Gateway.
- [x] Реализиране на механизъм за одит и логване (`audit_logger.py`).

## 🔵 Фаза 2: Усъвършенстван AI и MLOps (Завършена)
- [x] Имплементиране на AutoML енджин за автоматизирано обучение.
- [x] Създаване на пълноценен MLOps конвейер с MLflow интеграция.
- [x] Реализиране на XAI (Explainable AI) за интерпретируемост на моделите.
- [x] Разработване на модули за AI етика и безопасност.
- [x] Автоматизирано етикетиране на набори от данни.

## 🟡 Фаза 3: Индустрия 4.0 и Качествен Контрол (Текуща / Завършена)
- [x] Имплементиране на Статистически Процесен Контрол (SPC).
- [x] Разработване на енджин за Дигитални Двойници с предсказваща поддръжка.
- [x] Създаване на специализиран Dash GUI за качествен контрол в автомобилната индустрия.
- [x] Интеграция на FMEA (Failure Mode and Effects Analysis) в интерфейса.
- [x] Моделиране на адаптивни контролни цикли (PID, Fuzzy Logic).

## 🟠 Фаза 4: Устойчивост и Социално Въздействие (Завършена)
- [x] Разработване на тракер за въглероден отпечатък (Scopes 1, 2, 3).
- [x] Анализ на водното потребление и екологичното въздействие.
- [x] Симулация на общественото въздействие от AI автоматизацията.
- [x] Анализ на социалното поведение и настроения.

## 🔴 Фаза 5: Бъдещи Разширения (Текуща / Частично Завършена)
- [x] **Квантова Интеграция**: Реално използване на математически модели и квантови алгоритми (QUBO, QAOA, VQE) за оптимизация на веригата на доставки (завършено в v1.2.1/1.2.2).
- [x] **Edge AI Deployment**: Реализиране на Edge AI Inference Gateway с възможности за зареждане на модели, оптимизация, офлайн режим и бенчмарк (завършено в v1.2.2).
- [ ] **Разширена AR/VR функционалност**: Пълна интеграция с индустриални очила за добавена реалност за отдалечена поддръжка.
- [ ] **6G Комуникационен Протокол**: Дефиниране и тестване на протоколи за ултра-ниска латентност.
- [ ] **Автономен "Завод на Бъдещето"**: Пълна оркестрация на роботизирани линии без човешка намеса (Self-healing systems).
